Projects

The Silent Artery(Non-invasive Health Monitoring System)

Data Extraction

Projects related diagrams

This project presents a comprehensive analysis of biomedical parameter extraction using the MAX30102 sensor.

It explores the fundamental principles and mathematical foundations behind the extraction of vital health metrics including heart rate, blood oxygen saturation (SpO2), heart rate variability (HRV), perfusion index, and respiratory rate.

The implementation is based on the ESP32 microcontroller, enabling real-time signal processing and cloud connectivity through Google Cloud Platform (GCP) for efficient data storage and transmission.

The goal of this work is to demonstrate the feasibility of creating a robust, non-invasive health monitoring system suitable for both clinical and personal use.

By leveraging optical PPG signals captured via red and infrared light, the project effectively extracts and processes cardiovascular and respiratory signals in a compact embedded system.

This approach contributes to the growing domain of wearable health technology and remote patient monitoring systems.
